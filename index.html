Micro Interaction

Hovering or focusing on this element triggers a subtle animation that makes the button bounce slightly. This is done through a simple scale translation over 300ms with a distance of only 2px.

Timing Comparison

Timing Function 250ms

Timing Function 400ms

In this section, both buttons use a cubic-bezier timing function that creates a bouncy effect. The left button has a faster, more energetic bounce with a 250ms duration. While the right button uses a 400ms duration for a smoother, more gradual animation. The difference in timing generates a different feel for each despite using the same hover effect.
